Metrics

Moleculer has a built-in metrics feature. It can be enabled within the broker options with metrics: true option.
If it is enabled, the broker emits metrics events at every requests. You can subscribe to these events in your services and transfer them to your Tracer system (ZipKin, OpenTracingâ€¦etc).

Context params & meta

You can customize that the broker puts some ctx.meta and ctx.params fields to the metrics events.
