  • Blazing fast

    High-performance microservices framework. Check the benchmark results!

  • Extensible

    All built-in modules (caching, serializer, transporter) are pluggable. Create your own solutions or find all modules.

  • Open source

    Moleculer is a 100% open source and free of charge project. All source code is available under the MIT License on GitHub.

  • Fault tolerance

    Moleculer is a fault tolerant framework. It has built-in load balancer, circuit breaker, retries, timeout and bulkhead features.
